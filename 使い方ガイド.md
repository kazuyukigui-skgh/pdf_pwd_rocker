# PDFに鍵をかけるツール　使い方ガイド

**70代の方でも安心して使える、かんたんガイドです**

---

## 📱 このツールは何をするものですか？

ファイルに**パスワード（鍵）をかける**ツールです。

- 患者さんの情報など、大切な書類を守ることができます
- 鍵をかけたファイルは、パスワードを知っている人だけが開けます
- すべてパソコンの中で処理されるので、情報が外に出ることはありません

### 🆕 Word、Excel、PowerPointも使えます！

このツールは以下の形式に対応しています：

- 📝 **Word文書** (.docx) - 報告書、診断書など
- 📊 **Excel表** (.xlsx) - 患者リスト、データなど
- 📽️ **PowerPoint資料** (.pptx) - 研修資料、プレゼンなど
- 📄 **PDFファイル** (.pdf) - あらゆるPDF書類

→ **自動的にPDFに変換**してから鍵をかけるので、安心です

---

## 🚀 始める前に

### 必要なもの

1. ✅ Windowsパソコン（Windows 10 または 11）
2. ✅ 鍵をかけたいファイル（PDF、Word、Excel、PowerPoint）
3. ✅ パソコンのデスクトップ（作業する場所）

### 準備

- デスクトップに「PDF_Locker.exe」というファイルがあることを確認してください
- 鍵をかけたいファイルは「どこにあるか」覚えておいてください
  - 例：デスクトップ、マイドキュメント、など
- **Word/Excel/PowerPointを使う場合**:
  - Microsoft Officeがインストールされていることを確認してください
  - ファイルが開かれていない状態にしてください

## 📖 使い方（3つのステップ）

このツールは**3つのステップ**で使います。
画面の上に「①PDFを選ぶ」「②パスワードを決める」「③完了」と表示されます。

---

## ステップ① ファイルを選ぶ

### 1-1. ツールを起動する

1. デスクトップの「**PDF_Locker.exe**」を**ダブルクリック**します
   - ダブルクリック = 2回続けて速くクリック

2. セキュリティの警告が出たら「**実行**」または「**はい**」をクリックします

3. 画面が開きます
   - タイトル：「🔒 PDFに鍵をかけるツール」
   - 上に3つのステップが表示されます
   - 「①PDFを選ぶ」が緑色になっています

### 1-2. ファイルを選ぶ

1. 画面の真ん中に大きな**緑色のボタン**があります
   ```
   📁 ファイルを選ぶ
   ```

2. このボタンを**クリック**します

3. ファイルを選ぶ画面が開きます
   - 左側から場所を選びます（デスクトップ、ドキュメント、など）
   - 右側にファイルの一覧が表示されます

4. 鍵をかけたい**ファイル**を**クリック**して選びます
   - **PDF、Word、Excel、PowerPointが選べます**
   - 複数のファイルを選ぶこともできます
   - 複数選ぶには：**Ctrl キーを押しながら**クリック

5. 画面の右下にある「**開く**」ボタンをクリックします

6. メッセージが表示されます
   ```
   ○個のファイルを選びました。
   「次へ」ボタンを押してください。
   ```

7. 「**OK**」をクリックします

### 💡 ワンポイント

選んだファイルにアイコンが付きます：
- 📄 PDFファイル
- 📝 Word文書
- 📊 Excel表
- 📽️ PowerPoint資料

→ Word/Excel/PowerPointは**自動的にPDFに変換**されてから鍵がかかります

### 1-3. 次のステップへ

1. 画面の右下に青い「**次へ ▶**」ボタンが表示されます

2. このボタンを**クリック**します

3. ステップ②の画面に変わります

---

## ステップ② パスワードを決める

### 2-1. パスワードを入力する

1. 画面に大きな入力欄があります
   ```
   パスワード（4文字以上）:
   ［　　　　　　　　　　　］
   ```

2. この入力欄を**クリック**します

3. キーボードでパスワードを**入力**します
   - 4文字以上にしてください
   - 例：`byouin2024`、`kanja123`、など
   - 入力した文字は「●●●●」と表示されます（隠されます）

### 2-2. パスワードを確認する（オプション）

パスワードが見えないので心配な場合：

1. 入力欄の下にある「**パスワードを表示する**」に✓を入れます

2. パスワードが文字で見えるようになります

3. 確認したら、✓を外せばまた隠れます

### 2-3. パスワードをメモする

⚠️ **とても大切です！**

```
パスワードは忘れないようにメモしてください
パスワードを忘れるとPDFが開けなくなります
```

- **紙にメモ**して、安全な場所に保管してください
- パソコンのメモ帳などに保存しても良いです

### 2-4. 鍵をかける

1. パスワードを入力したら、右下の緑のボタンをクリックします
   ```
   鍵をかける ✓
   ```

2. 確認メッセージが表示されます
   ```
   このパスワードで鍵をかけます:
   「○○○○」
   よろしいですか？
   ```

3. パスワードが正しければ「**はい**」をクリックします
   - 間違っていたら「**いいえ**」を押して、入力し直します

### 2-5. 処理を待つ

1. 青い進行バーが表示されます
   ```
   処理中: ファイル名.pdf
   ▓▓▓▓▓▓▓▓▓▓▓▓▓
   ```

2. 少し待ちます（数秒～数十秒）
   - ファイルの数が多いと時間がかかります
   - **画面を閉じないでください**

3. 処理が終わると自動的に次の画面になります

---

## ステップ③ 完了

### 3-1. 完了画面

1. 画面に大きな✅マークが表示されます
   ```
   ✅
   鍵をかけ終わりました！
   ```

2. 保存した場所が表示されます
   ```
   保存した場所:
   C:\Users\あなたの名前\Desktop\パスワード付きPDF

   ファイル名の最初に「鍵付き_」が付いています。
   ```

### 3-2. ファイルを確認する

1. 画面の青いボタンをクリックします
   ```
   📁 保存した場所を開く
   ```

2. デスクトップの「**パスワード付きPDF**」フォルダが開きます

3. 中に「**鍵付き_○○.pdf**」というファイルがあります
   - これが鍵をかけたPDFファイルです

### 3-3. 試しに開いてみる

1. 「鍵付き_○○.pdf」を**ダブルクリック**します

2. パスワードを聞かれます
   ```
   このファイルはパスワードで保護されています
   パスワードを入力してください
   ```

3. さっき決めた**パスワードを入力**します

4. PDFが開けば成功です！✅

### 3-4. ツールを終わる

完了画面で、下のボタンをクリックします：

- 「**終了**」：ツールを閉じます
- 「**もう一度やる**」：最初から別のファイルに鍵をかけられます

## ❓ よくある質問

### Q1. パスワードを忘れてしまいました

**A:** 残念ながら、パスワードを忘れるとPDFは開けません。
- パスワードは必ず**紙にメモ**して保管してください
- 元のPDFファイル（鍵をかける前のファイル）は残しておくと安心です

### Q2. 元のPDFファイルはどうなりますか？

**A:** 元のファイルはそのまま残ります。
- 新しく「鍵付き_」というファイルが作られます
- 元のファイルは鍵がかかっていません
- 必要なら元のファイルは削除してOKです

### Q3. 保存先のフォルダはどこですか？

**A:** デスクトップの「パスワード付きPDF」フォルダです。
- 自動的に作られます
- 場所：デスクトップを開いて、「パスワード付きPDF」というフォルダを探してください

### Q4. エラーが出ました

**A:** よくあるエラーと対処法：

- 「**すでに鍵がかかっています**」
  → そのPDFはすでにパスワードがかかっています。別のファイルを選んでください

- 「**このファイルは開けません（使用中の可能性）**」
  → PDFファイルを開いていたら、閉じてから再度試してください

- 「**PDFファイルが壊れているかもしれません**」
  → ファイルが壊れています。別のファイルを選んでください

### Q5. 「次へ」ボタンが押せません

**A:** PDFファイルを選んでいない可能性があります。
- まず「📁 PDFファイルを選ぶ」ボタンを押してください
- ファイルを選ぶと「次へ」ボタンが押せるようになります

### Q6. 複数のファイルに一度に鍵をかけられますか？

**A:** はい、できます。
- ステップ①で**Ctrl キーを押しながら**複数のファイルをクリックして選びます
- すべてのファイルに同じパスワードがかかります

### Q7. WordやExcelファイルも使えますか？

**A:** はい、使えます！
- Word文書 (.docx)
- Excel表 (.xlsx)
- PowerPoint資料 (.pptx)
- PDFファイル (.pdf)

→ 自動的にPDFに変換してから鍵をかけます

**注意**：Microsoft Officeがインストールされている必要があります

### Q8. Word/Excelファイルで「変換に失敗」と出ます

**A:** 以下を確認してください：
1. **Microsoft Officeがインストールされているか**
   - Word、Excel、PowerPointがパソコンに入っている必要があります
2. **そのファイルが開かれていないか**
   - Word/Excelでファイルを開いていたら、閉じてから再度試してください
3. **ファイルが壊れていないか**
   - 一度そのファイルをダブルクリックして、正常に開けるか確認してください
